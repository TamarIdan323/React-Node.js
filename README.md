# React-Node.js

# אתר קניות מוצרים

פרויקט זה הוא אתר קניות מבוסס React ו-Node.js, עם מסד נתונים MongoDB.

## תכונות עיקריות

- הרשמה וכניסה למשתמשים  
- סל קניות דינמי וניהול הזמנות  
- מערכת תשלום מאובטחת  
- ממשק ניהול לניהול מוצרים והזמנות עבור מנהלים  

## טכנולוגיות בשימוש

- React (Frontend)  
- Node.js עם Express (Backend)  
- MongoDB (בסיס נתונים)
